# 테트리스 — War Zone Edition

HTML, CSS, JavaScript만 사용하는 브라우저 테트리스 게임 교육용 프로젝트입니다.  
QHD(2560×1440) 풀스크린 전쟁 배경과 Cursor를 활용한 개발·리뷰·리팩토링을 진행합니다.

## 실행 방법

1. 이 프로젝트 폴더를 연다.
2. `index.html` 파일을 더블클릭하거나, 브라우저로 드래그한다.
3. 또는 브라우저 주소창에 파일 경로를 입력한다.

   ```
   file:///C:/DEV/tetris-cursor-main/index.html
   ```

별도의 빌드 도구나 서버 설치 없이 바로 실행할 수 있습니다.

## 파일 구조

```
tetris-cursor-main/
├── index.html   # 게임 화면 + 전쟁 배경 + 킹콩 스테이지
├── style.css    # QHD 풀스크린 스타일 및 전쟁 애니메이션
├── script.js    # 게임 로직
├── 프롬프트.txt  # Cursor 리뷰/리팩토링 프롬프트
└── README.md    # 프로젝트 안내
```

## 현재 상태

- 게임 보드, 점수, 시작/재시작 버튼, 조작법 안내 UI
- 블록 생성, 자동 낙하, 고정, 라인 삭제, 점수, 게임 오버 처리
- 키보드 조작 (이동, 회전, 소프트/하드 드롭)
- 회전 시 벽 킥(wall kick) 지원
- QHD 풀스크린 전쟁 배경 (폭발, 연기, 탄환 궤적, 탐조등, 레이더 HUD)
- 군사 HUD 스타일 UI + 킹콩 댄스 애니메이션
- 블록 모양별 개별 색상 적용
- 라인 클리어 전투 이펙트 (도미노 삭제, 미사일, 폭격기, 대형 폭발)
- 탄약 예고(다음 블록 3개), Hold(C), 고스트 피스, 폭탄 블록
- ARTILLERY COMBO, 킹콩 반응 연출, Web Audio 효과음
- 레벨 시스템, 위험 경고, localStorage 최고 기록
- P 키 일시 정지

## 라인 클리어 효과

| 조건 | 효과 |
|---|---|
| 1줄 | 플래시 후 즉시 삭제 |
| 2줄 | 상단 미사일 낙하 + 도미노식 순차 삭제 |
| 3줄 이상 | 비행기 폭탄 투하 + 대형 폭발 + 위·아래 줄 연쇄 삭제 + 도미노 삭제 |

## 블록 색상

| 블록 | 모양 | 색상 |
|---|---|---|
| I | 긴 막대 (ㅡ) | 노란색 |
| O | 정사각형 (ㅁ) | 금색 |
| T | T자 | 보라색 |
| S | S자 (지그) | 초록색 |
| Z | Z자 (역지그) | 빨간색 |
| J | 반전 기역자 (ㅓ) | 파란색 |
| L | 기역자 (ㅏ) | 주황-빨강 |

## 조작법

게임이 진행 중일 때만 키 입력이 적용됩니다. 충돌이 발생하면 이동·회전은 취소됩니다.

| 키 | 동작 |
|---|---|
| `ArrowLeft` (←) | 왼쪽으로 한 칸 이동 |
| `ArrowRight` (→) | 오른쪽으로 한 칸 이동 |
| `ArrowDown` (↓) | 한 칸 빠르게 내리기 |
| `ArrowUp` (↑) | 블록 회전 (벽·블록과 충돌 시 취소, 벽 킥 적용) |
| `Space` | 즉시 바닥까지 낙하 (hard drop) |
| `C` | 블록 보류 (Hold) |
| `P` | 일시 정지 / 재개 |

## 변경 이력

### v6 — 게임플레이 강화 & 일시 정지

- **탄약 예고**: 다음 블록 3개 HUD 미리보기
- **Hold**: `C` 키로 블록 보류·교환
- **고스트 피스**: 착지 예상 위치 반투명 표시
- **폭탄 블록**: 8% 확률 등장, 고정 시 주변 3×3 제거
- **ARTILLERY COMBO**: 연속·다줄 클리어 시 배너 및 점수 배율
- **킹콩 반응**: 2줄 점프/박수, 3줄+ 격한 춤 + ROAR!
- **효과음**: 미사일·폭격기·도미노·폭탄·콤보·킹콩 (Web Audio API)
- **레벨**: 10줄마다 레벨 UP, 낙하 속도 증가
- **위험 경고**: 상단 절반에 블록 시 보드 테두리 빨간 점멸
- **최고 기록**: localStorage에 최고 점수·레벨 저장
- **일시 정지**: `P` 키로 전투 일시 중단 오버레이

### v5 — 라인 클리어 전투 이펙트

- **2줄 이상**: 도미노식 단계별 라인 삭제 애니메이션
- **2줄**: 상단에서 미사일 낙하 및 폭발 이펙트
- **3줄 이상**: 비행기 횡단 폭탄 투하, 대형 폭발, 화면 흔들림
- **3줄 이상 보너스**: 완성 줄의 위·아래 줄 연쇄 삭제
- 이펙트 재생 중 입력·자동 낙하 일시 정지

### v4 — 블록 색상 구분

- 테트로미노 7종 모양별 개별 색상 적용 (긴막대 노랑, 기역자 주황-빨강 등)
- `script.js` PIECES 정의에 모양별 주석 추가

### v3 — War Zone Edition (QHD 풀스크린)

- **레이아웃**: `100vw × 100vh` 풀스크린 구성, 3열 그리드(좌측 전장 · 중앙 보드 · 우측 HUD)
- **QHD 대응**: 셀 크기 `clamp(30px, (100vh - 140px) / 20, 44px)` — 2560×1440에서 최대 44px
- **전쟁 배경**: 폭발, 연기, 탄환 궤적, 조명탄, 탐조등, 폐허·전차·헬기 실루엣, 비네팅·필름 그레인
- **UI**: 군사 HUD 스타일(앰버/올리브), 코너 브래킷, 레이더 위젯, 상태 티커
- **킹콩**: 좌측 SECTOR ALPHA 플랭크에 댄스 애니메이션 유지
- **반응형**: 1100px 이하 좌측 플랭크 숨김, 600px 이하 단일 열 레이아웃
