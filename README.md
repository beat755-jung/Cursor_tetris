# 테트리스 — War Zone Edition

HTML, CSS, JavaScript만 사용하는 브라우저 테트리스 게임 교육용 프로젝트입니다.  
QHD(2560×1440) 풀스크린 전쟁 배경과 Cursor를 활용한 개발·리뷰·리팩토링을 진행합니다.

## 실행 방법

1. 이 프로젝트 폴더를 연다.
2. `index.html` 파일을 더블클릭하거나, 브라우저로 드래그한다.
3. 또는 브라우저 주소창에 파일 경로를 입력한다.

   ```
   file:///C:/DEV/tetris-cursor-main/index.html
   ```

별도의 빌드 도구나 서버 설치 없이 바로 실행할 수 있습니다.

## 파일 구조

```
tetris-cursor-main/
├── index.html   # 게임 화면 + 전쟁 배경 + 킹콩 스테이지
├── style.css    # QHD 풀스크린 스타일 및 전쟁 애니메이션
├── script.js    # 게임 로직
├── 프롬프트.txt  # Cursor 리뷰/리팩토링 프롬프트
└── README.md    # 프로젝트 안내
```

## 현재 상태

- 게임 보드, 점수, 시작/재시작 버튼, 조작법 안내 UI
- 블록 생성, 자동 낙하, 고정, 라인 삭제, 점수, 게임 오버 처리
- 키보드 조작 (이동, 회전, 소프트/하드 드롭)
- 회전 시 벽 킥(wall kick) 지원
- QHD 풀스크린 전쟁 배경 (폭발, 연기, 탄환 궤적, 탐조등, 레이더 HUD)
- 군사 HUD 스타일 UI + 킹콩 댄스 애니메이션

## 조작법

게임이 진행 중일 때만 키 입력이 적용됩니다. 충돌이 발생하면 이동·회전은 취소됩니다.

| 키 | 동작 |
|---|---|
| `ArrowLeft` (←) | 왼쪽으로 한 칸 이동 |
| `ArrowRight` (→) | 오른쪽으로 한 칸 이동 |
| `ArrowDown` (↓) | 한 칸 빠르게 내리기 |
| `ArrowUp` (↑) | 블록 회전 (벽·블록과 충돌 시 취소, 벽 킥 적용) |
| `Space` | 즉시 바닥까지 낙하 (hard drop) |

## 변경 이력

### v3 — War Zone Edition (QHD 풀스크린)

- **레이아웃**: `100vw × 100vh` 풀스크린 구성, 3열 그리드(좌측 전장 · 중앙 보드 · 우측 HUD)
- **QHD 대응**: 셀 크기 `clamp(30px, (100vh - 140px) / 20, 44px)` — 2560×1440에서 최대 44px
- **전쟁 배경**: 폭발, 연기, 탄환 궤적, 조명탄, 탐조등, 폐허·전차·헬기 실루엣, 비네팅·필름 그레인
- **UI**: 군사 HUD 스타일(앰버/올리브), 코너 브래킷, 레이더 위젯, 상태 티커
- **킹콩**: 좌측 SECTOR ALPHA 플랭크에 댄스 애니메이션 유지
- **반응형**: 1100px 이하 좌측 플랭크 숨김, 600px 이하 단일 열 레이아웃
